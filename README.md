# LANtarian 🌐

Projeto de site moderno para uma "Lana House" digital chamada LANtarian.

## 🚀 Sobre o projeto

LANtarian é uma landing page fictícia com estilo futurista, simulando uma casa inteligente conectada com tecnologia e automação.

## 🧠 Tecnologias

- HTML5
- CSS3
- JavaScript

## 🎯 Objetivo

Criar uma base inicial profissional para evolução do projeto para um sistema completo de casa inteligente e automação digital.

## 💡 Próximos passos

- Melhorar UI/UX
- Adicionar animações
- Criar painel de controle (dashboard)
- Integrar backend
